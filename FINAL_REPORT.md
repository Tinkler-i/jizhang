## 💡 快速开始

### 一键启动（3步）

```bash
# 1. 启动后端（在项目根目录）
mvn spring-boot:run

# 2. 启动前端开发服务器（新终端窗口）
cd src/main/resources/static/app
npm install
npm run dev

# 3. 打开浏览器访问
http://localhost:5173
```

### 生产部署

```bash
# 1. 构建前端
cd src/main/resources/static/app
npm run build

# 2. 打包 Spring Boot 应用
mvn clean package

# 3. 运行应用
java -jar target/jizhang-1.0.0.jar

# 4. 访问应用
http://localhost:8080/jizhang/
```
