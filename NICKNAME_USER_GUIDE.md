## 🆕 新用户注册

### 注册流程
1. 进入系统登录页面
2. 点击"注册"链接
3. 填写注册信息：
   - 用户名 (如: `xiaoming`)
   - 邮箱
   - 密码
   - 验证码
4. 点击"注册"按钮

### 注册完成后
- ✅ 系统自动为你创建昵称（初始值 = 用户名）
- ✅ 自动创建个人家庭组
- ✅ 自动成为家庭组管理员

> **提示**: 注册完成后，你可以在"个人设置"中修改昵称

## 📝 修改昵称

### 方式 1️⃣: 网页端修改

#### 步骤
1. 点击右上角"个人设置"或用户菜单
2. 进入"个人设置"页面
3. 找到"基本信息"卡片
4. 看到四个字段：
   - 用户名（不可修改）
   - **昵称** ← 在这里修改
   - 邮箱
   - 电话
5. 在"昵称"输入框中输入新昵称
6. 点击"保存修改"按钮

#### 示例
```
修改前:
昵称: xiaoming

修改后:
昵称: 小明
```

### 方式 2️⃣: 直接调用 API

#### 使用 curl
```bash
curl -X PUT http://localhost:8080/api/user/nickname \
  -H "Content-Type: application/json" \
  -d '{"nickname": "小明"}'
```

#### 使用 JavaScript
```javascript
async function updateNickname(newNickname) {
  const response = await fetch('/api/user/nickname', {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ nickname: newNickname })
  });
  
  const result = await response.json();
  console.log('结果:', result);
}

// 调用
updateNickname('小明');
```

### 昵称规则
| 规则 | 说明 |
|------|------|
| **不能为空** | 昵称必须输入 |
| **不能只有空格** | "   " 不合法 |
| **最长 50 字符** | 不超过 50 个汉字或字符 |
| **保存立即生效** | 修改后立即更新 |

### 昵称修改验证

如果输入错误，会显示红色错误提示：

```
❌ 昵称不能只包含空格
❌ 昵称不超过 50 个字符
```

此时"保存修改"按钮会被禁用。修复错误后按钮恢复可用。

### 重置昵称

如果你修改了昵称但想恢复，点击"重置昵称"按钮：

```
修改前: 小明
输入新值: 小红
点击"重置昵称"
恢复为: 小明
```

## 👥 查看家庭成员

### 在家庭组中查看成员昵称

#### 操作步骤
1. 点击左侧导航菜单"家庭管理"
2. 选择"成员管理"标签
3. 可以看到家庭组中所有成员

#### 成员列表显示

**修改前:**
```
[管理员] 用户 1
[成员]   用户 2
[成员]   用户 3
```

**修改后:**
```
[管理员] 小明
         @xiaoming
         [权限] [移除]

[成员]   小红
         @xiaohong
         [权限] [移除]

[成员]   小王
         @xiaowang
```

### 理解成员信息

| 信息 | 说明 | 示例 |
|------|------|------|
| **角色徽章** | ADMIN 或 MEMBER | [管理员] |
| **昵称** | 用户自定义的显示名 | 小明 |
| **用户名** | 注册时的用户名 | @xiaoming |
| **操作按钮** | 权限和移除选项 | [权限] [移除] |

### 权限管理

如果你是家庭组管理员，可以：

1. 点击某个成员的"权限"按钮
2. 在弹出的对话框中看到成员昵称
3. 修改权限模板或角色

```
修改成员权限
小明 (@xiaoming)

权限模板: [记账员 v]
角色: [成员 v]

[取消] [保存]
```

## ❓ 常见问题

### Q1: 为什么注册后昵称是用户名？

**答**: 系统默认将昵称设置为用户名，方便你快速开始使用。你可以随时在"个人设置"中修改昵称。

### Q2: 修改昵称后，其他成员什么时候能看到？

**答**: 修改立即保存到数据库，其他成员下次打开"成员管理"页面时就能看到最新的昵称。无需刷新或重新登录。

### Q3: 能删除昵称吗？

**答**: 不能。昵称必须有值（不能为空），如果想恢复默认值，可以：
1. 在昵称框输入你的用户名
2. 点击"保存修改"

### Q4: 昵称能包含特殊字符吗？

**答**: 可以。昵称支持任何字符，但不超过 50 个字符。支持：
- ✅ 中文: "小明"
- ✅ 英文: "Xiaoming"
- ✅ 数字: "123"
- ✅ 表情符号: "小明😊"（某些浏览器可能不支持）
- ✅ 特殊符号: "小明-王"

### Q5: 两个人能有相同的昵称吗？

**答**: 可以。系统通过用户名区分用户，昵称可以重复。同一家庭组中会显示：
```
小明 (@user1)
小明 (@user2)  ← 不同的用户名
```

### Q6: 修改昵称需要什么权限？

**答**: 
- 用户只能修改自己的昵称
- 无需特殊权限
- 任何登录用户都可以修改
- 家庭组管理员无法强制修改成员昵称

### Q7: 昵称修改历史能看到吗？

**答**: 目前不支持。系统只保存最新的昵称，不记录修改历史。如需历史记录，请联系管理员。

### Q8: 退出登录后昵称会变吗？

**答**: 不会。昵称保存在数据库中，与登录状态无关。重新登录后昵称保持不变。

### Q9: 怎样快速修改多个成员的权限？

**答**: 
1. 打开"成员管理"
2. 对每个成员点击"权限"按钮
3. 在弹出对话框中修改权限
4. 点击"保存"

暂不支持批量修改，需要逐个操作。

### Q10: 忘记了自己的用户名怎么办？

**答**: 用户名显示在：
1. "个人设置"页面，用户名字段（不可修改）
2. 成员列表中，@符号后的名字
3. 邮件或其他系统通知中

### Q11: 修改昵称时出错怎么办？

**答**: 
1. 检查网络连接
2. 确保已登录
3. 检查昵称是否满足规则：
   - 不为空
   - 不超过 50 字符
4. 如仍有问题，请刷新页面重试
5. 或联系系统管理员

### Q12: 能否导出成员列表（包括昵称）？

**答**: 暂不支持直接导出，但可以：
1. 截图保存
2. 复制粘贴到 Excel
3. 或联系管理员获取数据导出
